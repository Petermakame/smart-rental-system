RELAY WIRING DETAIL:
────────────────────
  12V PSU (+) ──────────────── Solenoid Lock (+)
  12V PSU (-) ──────────────── Relay GND
  
  Relay COM ─────────────────── 12V PSU (-)  [Common]
  Relay NO  ─────────────────── Solenoid Lock (-)  [Normally Open]
  
  ┌──────────────────────────────────────────────────────────┐
  │  When Relay fires (ESP32 sends LOW to Pin 26):           │
  │    NO → COM connects → Current flows → Lock OPENS        │
  │  When Relay off (ESP32 sends HIGH or no signal):         │
  │    NO → COM breaks → No current → Lock CLOSES (locked)  │
  └──────────────────────────────────────────────────────────┘

FULL POWER DIAGRAM:
───────────────────
  [USB 5V] → ESP32 (powers ESP32 + RFID + Fingerprint)
  [12V Adapter] → Solenoid Lock (via Relay)
  [5V from ESP32 Vin] → Relay Module VCC
